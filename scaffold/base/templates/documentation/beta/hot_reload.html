Hot reload

This feature should be enabled while you are developing a project. Before µWeb had to be manualy restarted every time you made a change in a file. While this feature is enabled it watches all of your µWeb files and restarts when something changes.

To make use of this feature all you have to do is add the following line to your config.ini:

[development]
dev = True
