Automatic route detection

Changing the route directory

If you want µWeb3 to scan a different directory than the default directory we can change this in the config. Navigate to:

project_folder/scaffold/base/config.ini

and add the following key to the routing section:

[routing]
default_routing = new_directory

Disabling automatic route detection

The automatic route detection is enabled by default in µWeb3. To disable this setting naviage to:

project_folder/scaffold/base/config.ini

and add the following key to the routing section:

[routing]
disable_automatic_route_detection = True

Manually adding routes

If you rather not use the automatic route detection feature you can also manually add routes.

To do this navigate to: /project_folder/scaffold/base/__init__.py. In this file you can modify the following line of code to adjust the routes you want to use.

return uweb3.uWeb([pages.PageMaker, your_route.PageMaker], routes, config=config)

Or for a simple application with a single route:

return uweb3.uWeb(pages.PageMaker, routes, config=config)
