Event listener

µWeb3 also supports creating/deleting/listening to events. µWeb3 invokes events at specific times in the application. When one of those events is invoked the event listener calls the functions listening to the event. This should give the user more controll over the application without having to dive into the source code.

Adding an event to an event listener

Lets start with creating a function that we can supply to our event listener: 

 1import uweb3
 2from uweb3 import globals
 3
 4def HelloWorld():
 5print("Hello World!")

Now that we have our function we can add it to an event and invoke it: 

 1import uweb3
 2from uweb3 import globals
 3
 4def HelloWorld():
 5print("Hello World!")
 6
 7def RouteHandler(self):
 8def Route(self):
 9globals.event_listener.add("my_event", HelloWorld)
10globals.event_listener("my_event") #Invoking the event
>>> Hello World!
