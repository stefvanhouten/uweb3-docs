Getting started

Running µWeb3

Now that we have our virtual environment setup and our µWeb3 installed it is time to create our first application. First lets download the a file which has all the basic structure set up for us. Click here to download the file. Copy the downloaded file into your project folder to get started.

To run the application all we need to do now is the following: 

>>> cd project_folder/scaffold
>>> python3 serve.py
Our application is now available on: 127.0.0.1:8000
