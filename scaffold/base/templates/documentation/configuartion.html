Configuartion

Basic configuartion µWeb3

Configuarting µWeb3 is as easy as changing some settings in a config file. Lets get started. Navigate and open project_folder/scaffold/base/config.ini.

All settings in this file are accessible in any µWeb3 pagemaker file, more about the pagemaker later, for now all we need to know that is that we should store our settings here.

A few default settings are: 

[development]
dev = True
Setting the dev flag to True can be use for many things, one of those is that µWeb3 is now monitoring your files for changes and will restart accordingly.

Database configuartion µWeb3

In the same config file as above we also setup our database configuration. With this information µWeb3 creates a connection object which is available depending on the pagemaker that you use. 

[mysql] OR [sqlite]
host = host
user = username
password = pass
database = dbname
