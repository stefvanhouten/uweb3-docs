Installation

Setting up the virtual environment

The easiest and quickest way to install µWeb3 is using Python's virtualenv.

Create a project folder and a virtual environment folder:

>>> mkdir project_folder
>>> cd project_folder

>>> virtualenv uweb3-venv
Activate the environment
>>> source uweb3-venv/bin/activate

uWeb3 Dependencies

uWeb3 makes use of the following packages:

  • decorator
  • PyMySQL
  • python-magic
  • python3-openid
  • pytz
  • simplejson
  • sqlalchemy
  • werkzeug
  • mysqlclient

Installation

µWeb3 can be installed with pip3.

>>> pip3 install uWeb3
